API 接入状态

小红书/抖音：不直接接入 用素材导入替代平台抓取 首版不依赖平台 API 或抓取账号，采用手动整理/结构化导入热门话题和文案信号。

长期维护

运营维护中心

内容更新、官方复核、素材导入和发布留档都在这里收束，方便后续长期维护。
官方信息复核：出发前必做 开放、预约、票务、交通和临时闭馆 每次出发或发布前，优先复核官方渠道和现场变化，避免过期信息影响使用。
平台素材更新：结构化导入 热点话题、游记碎片和平台文案信号 采用人工整理或素材导入，不直接抓取平台账号数据，保留来源和复核边界。
路线地图巡检：新增路线后复核 首页示意图、详情页路线图和 GeoJSON 新增或调整路线后，检查点位顺序、换区成本、地图标记和真实地图增强入口。
发布与备份：更新后留档 静态站点、离线包、部署仓库和验收截图 当前覆盖 6 条路线；更新后重新生成站点并保留可回滚产物。
