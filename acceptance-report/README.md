# 上海攻略生成工作台公开验收报告包

这个包用于把当前发布版本的验收口径交给未来维护者、协作者或试用方。它记录应该看哪些测试、哪些发布入口、哪些线上检查，以及哪些事实边界仍需人工复核。

## 文件

- `test-summary.json`：测试摘要和推荐复跑命令。
- `release-summary.md`：发布摘要、入口链接和交付物索引。
- `online-acceptance-checklist.md`：线上验收清单。
- `acceptance-manifest.json`：包内文件和源资产索引。

## 测试摘要

当前首版验收以 web workbench 回归、全量回归、线上 ZIP 下载和移动端视口检查为核心。

## 剩余边界

- 官方开放、预约、票务、交通和临时闭馆仍需出发前人工复核。
- 小红书、抖音和旅游平台素材只作为结构化灵感，不自动抓取、不复制原文、不自动发布。
- 高德地图、天气等 API 是后续增强，不是当前网页打开和交付验收的前置条件。
