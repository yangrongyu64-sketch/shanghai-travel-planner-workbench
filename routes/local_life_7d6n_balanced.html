吃什么 / 休息补给

静安寺 / 南京西路

住宿区晚餐、商场兜底、购物和城市质感路线

吃喝类型：商场正餐 / 咖啡 / 甜品 / 晚餐

休息策略：把高确定性的晚餐和雨天休息放在商圈里，适合抵达日、返程前和长住补给。

避坑：预算很紧或想找强烈街区烟火气时，不要把这里当唯一餐饮答案。

人民广场 / 南京东路

首次来上海、短途换乘、经典路线补给

吃喝类型：商场简餐 / 本帮简餐 / 咖啡 / 甜品

休息策略：把午餐或下午茶放在商场、地铁站周边，适合接博物馆、南京东路和外滩路线。

避坑：不要在午晚高峰临时追热门排队店，容易打乱后续外滩或返程节奏。

外滩 / 外滩源

外滩夜景前后的轻食、咖啡和建筑路线停顿

吃喝类型：咖啡 / 轻食 / 下午茶 / 简餐

休息策略：把正式用餐放在外滩源或南京东路一侧，江边只承担散步和拍摄，不把吃饭压在观景点上。

避坑：避免为江景位支付过高溢价，也不要在夜景高峰前临时排队。
