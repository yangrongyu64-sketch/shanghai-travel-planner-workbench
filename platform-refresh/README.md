# 上海攻略生成工作台平台素材刷新包

这个包用于把小红书、抖音、旅游网站等平台素材转成可复核、可导入、可持续更新的结构化输入。

## 核心原则

- 不复制第三方原文，只保留结构化洞察。
- 平台素材只作为路线结构、体验信号、标题角度和避坑线索。
- 开放时间、票价、预约、交通、临时闭馆等事实字段必须回到官方来源复核。

## 文件

- `capture-workflow.md`：从粘贴素材到合并入路线数据的命令流程。
- `intake-template.json`：每条路线的素材收件箱模板。
- `topic-signals.json`：按路线生成的小红书、抖音、旅游网站素材采集提示。
- `acceptance-checklist.md`：素材刷新验收清单。
- `platform-refresh-manifest.json`：包内文件和源资产索引。
