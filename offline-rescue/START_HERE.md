# START HERE：上海攻略离线应急包

这个包用于在公网链接、本机工作目录或 Codex 账号暂时不可用时，继续查看和恢复上海攻略生成工作台的核心内容。

## 立刻打开

1. 解压 `offline-rescue-kit.zip`。
2. 双击 `site/index.html`，进入普通游客首页。
3. 需要编辑、复制和下载内容时，打开 `site/workbench.html`。
4. 需要查地点库时，打开 `site/place_catalog.html`。
5. 需要单条路线时，打开 `site/routes/relaxed_2d1n_citywalk.html` 或其他路线页。

## 恢复个人行程状态

1. 如果你保存过 `shanghai-traveler-data-vault.json`，打开 `site/index.html`。
2. 找到“旅行数据保险箱”。
3. 粘贴或选择 `shanghai-traveler-data-vault.json`。
4. 点击导入备份，确认偏好、收藏、草稿和游记碎片恢复。

## 可对照文件

- `product-delivery/data-vault-recovery-guide.md`：完整恢复步骤。
- `release-evidence/data-vault-recovery-evidence.json`：恢复能力发布证据。
- `offline-rescue-checklist.md`：离线验收清单。
- `checksums.json`：关键文件 SHA256 校验。
