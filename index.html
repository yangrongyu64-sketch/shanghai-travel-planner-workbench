能力边界

API 接入状态

无需 API Key 也可正常使用；第三方 API 只做增强，不影响静态网页、离线包和普通路线查看。
高德地图：可选增强 路线详情页真实地图增强 普通使用不需要 Key；填写高德 Web Key 与安全密钥后，可在路线详情页加载真实地图。
Open-Meteo 天气：可选预检 免费无 Key 出行风险提示 首页可刷新未来 5 天上海天气；接口失败时继续使用静态风险规则。
腾讯地图：备选 小程序迁移时优先评估 如果转微信小程序，可把腾讯地图作为路线页地图能力的备选实现。
小红书/抖音：不直接接入 用素材导入替代平台抓取 首版不依赖平台 API 或抓取账号，采用手动整理/结构化导入热门话题和文案信号。
